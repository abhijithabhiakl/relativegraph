# Contributing to RelationTree

First off, thank you for considering contributing to RelationTree! It's people like you that make OSS great.

## Development Workflow

1. **Fork** the repo on GitHub
2. **Clone** the project to your own machine
3. **Commit** changes to your own branch
4. **Push** your work back up to your fork
5. Submit a **Pull Request** so that we can review your changes

## Code Structure

- `/src` contains the React Frontend and Reducer store. UI Styling uses standard CSS inside `src/styles`.
- `/server.js` acts as the simple Express backend REST server.
- `/prisma` contains the `schema.prisma` definition for the SQLite database layout.

## Submitting Pull Requests

- When creating your PR, please describe the problem you are solving comprehensively.
- Keep PRs focused on one goal.
- If you add new feature, please take a screenshot and include it in your PR so we can quickly understand the visual layout.
